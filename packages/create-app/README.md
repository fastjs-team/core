## Description

This is the offical CLI for Fastjs. For the core library, please visit [fastjs-team/core](https://github.com/fastjs-team/core).

## Start Using

```bash
pnpm create fastjs
```

```bash
yarn create fastjs
```

```bash
bun create fastjs
```

```bash
npm init fastjs
```
